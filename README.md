# LaTeXZero-Remake-Version
Remaked from [LaTeX Templates for Mathematical Notes](https://github.com/fenglielie/latexzero).

## Introduction
Several class options are designed: *lang*, *color*, *counter*, *mode* with original book class options passed. Please refer to the Preface of the document.

## Compiling Environment
TeXPage (online) or Windows 11 + TeX Live 2025 + VSCode Latex Workshop 2025. XeTeX and PdfLaTeX are both available for English documents but only XeTeX for Chinese documents.
